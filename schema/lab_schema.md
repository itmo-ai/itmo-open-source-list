# Схема карточки лаборатории

Каждая лаборатория описывается отдельным JSON-файлом в папке `labs/`.

## Обязательные поля

- **`id`** (string)  
  Уникальный идентификатор лаборатории латиницей. Используется для связи с проектами.

- **`name`** (string)  
  Название лаборатории (желательно на английском или транслитерация, так как используется как системное имя).

- **`description`** (object)  
  Описание деятельности лаборатории.
  - **`ru`** (string, **обязательно**) — описание на русском языке.
  - `en` (string, *опционально*) — описание на английском.

## Опциональные поля

- **`github_org`** (string, URL)  
  Ссылка на организацию в GitHub (если есть).

- **`links`** (object)  
  Ссылки:
  - `website`
  - `telegram`
  - `email`
  - `vkontakte`

- **`example_projects`** (array of strings)  
  Список названий ключевых проектов (для быстрого предпросмотра, если автоматическая связь через `projects/` ещё не построена).
