### Первый блок прямо перед главным(который отвечает за превью).
Никаких изменений в нем делать не нужно

### Второй блок для плашки, соответственно перед блоком с плашкой
Может в будущем потребоваться изменить точку фриза у плашки
делается в строке `      var stopStick = window.scrollY + cardsRect.bottom - groupHeight + 80;`
соответственно нужно менять последний численный параметр

### Третий блок после блоков (аналитика и научпоп+кроме того), для отображения фона
Также не требуется дополнять в дальнейшем

  #rec1945326471 .t396__artboard,

  }
